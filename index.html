Transform

A propriedade transform aplica uma transformação 2D ou 3D a um elemento.
Esta propriedade permite girar, dimensionar, mover, inclinar, etc., elementos.

Matriz 3d

A matriz 3D no CSS é usada para aplicar transformações tridimensionais a elementos HTML. Com a propriedade transform, é possível
usar a função matrix3d(), que aceita 16 valores em uma matriz 4x4. Esses valores definem rotações, escalas, translações e distorções
no espaço 3D. A sintaxe é: 

        transform: matrix3d(   |   a1, b1, c1, d1, a2, b2, c2, d2, a3, b3, c3, d3, d4
          a1, b1, c1, d1,      |   descrevem a transformação linear. enquanto 
          a2, b2, c2, d2,      |   a4, b4 e c4 descrevem a translação
          a3, b3, c3, d3,      | 
          a4, b4, c4, d4       |
        )
Entender a matriz 3d é importante pois a transformação aplicada no elemento é armazenada em matriz 3d no navegador, ou seja, quando você
quiser acessar e alterar uma transformação usando javascript, irá precisar manipular a matriz 3d daquele elemento
